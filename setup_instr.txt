SETUP INSTRUCTIONS
For users who will want to be able to run our code.

Usage:
cd into the recognize_digits folder

  python kmeans_raw.py k {method} {initialization ty}

  python kmeans_scikitlearn.py k
  where k is the number of clusters

Setup:
  Training/testing images should first be manually unzipped, and placed in same directory

Dependencies:
 CVXOPT; “pip install cvxopt --user”, else follow http://cvxopt.org/install/
 Numpy; “easy_install numpy”
